# 🌦️ WeathFly

**WeathFly** is a modern, UI-first weather web application that provides real-time weather data, air quality insights, and a smooth interactive experience.  
Built using vanilla JavaScript and public APIs, the focus is on clean design, usability, and performance

---

## ✨ Features

- 🌍 **Search weather by city**
- 🌡️ **Current temperature, min/max**
- 📆 **Hourly & multi-day forecast**
- 📊 **Interactive temperature graph**
- 🌫️ **Air Quality Index (AQI) with status**
- 🌅 **Sunrise & sunset timings**
- 🌙 **Day / Night UI behavior**
- ⚡ **Fast, lightweight, no frameworks**

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **OpenWeatherMap API**
- **SVG (for graphs & visuals)**

## 🚀 How It Works

1. User enters a city name  
2. App fetches weather + AQI data from OpenWeatherMap  
3. Data is processed and displayed in a clean dashboard  
4. Graph updates dynamically based on forecast timeline  

---

## 🔑 API Used

- [OpenWeatherMap](https://openweathermap.org/)
  - Current Weather
  - Forecast
  - Air Pollution (PM2.5 → AQI)
